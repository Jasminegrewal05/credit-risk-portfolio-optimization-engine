# Automated Credit Risk & Portfolio Optimization Engine

Developed an end-to-end credit risk management system on 30,000 credit card accounts using XGBoost-based Probability of Default (PD) modeling, reject inference, SHAP explainability, and automated credit line optimization.

## Features

- Feature Engineering
- Synthetic CIBIL Score
- Reject Inference
- XGBoost PD Modeling
- Hyperparameter Tuning
- Risk Segmentation
- Credit Line Increase/Decrease Decisioning
- Business Guardrails
- SHAP Explainability
- Streamlit Dashboard

## Model Performance

- ROC-AUC: 0.82
- Precision: 0.67
- Recall: 0.59
- F1 Score: 0.63

## Tech Stack

Python, Pandas, NumPy, XGBoost, SHAP, Streamlit, Plotly

## Dashboard

Portfolio monitoring, customer-level risk analysis, explainability, and credit limit optimization.
